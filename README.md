# EXIT 67 - 3D Horror Arcade Web Game

**EXIT 67**은 일본의 인기 인디 공포 게임인 **‘8번출구(Exit 8)’**를 모티브로 한 WebGL 기반 3D 미스터리 탈출 루프 게임입니다. 

플레이어는 기괴하고 끝없이 반복되는 지하 통로에서 탈출하여 최종 출구인 67번 출구에 도달해야 합니다. 복도 곳곳에서 발생하는 67가지의 기묘한 이상현상(이변)을 식별하고 올바른 판단을 내리십시오.

---

## 🎮 게임 규칙 (Rules)

1. **관찰**: 복도에 진입하면 주변의 포스터, 문, 소화전, 조명, 사운드 등 모든 요소를 주의 깊게 살핍니다.
2. **판단 & 선택**:
   * **이상현상이 전혀 없을 경우**: 망설임 없이 앞으로(Front Door) 전진하여 문을 열고 나갑니다.
   * **이상현상이 하나라도 발견될 경우**: 즉시 뒤로(Back Door) 돌아가 복도를 빠져나갑니다.
3. **루프**:
   * 올바르게 판단하면 출구 번호가 **1씩 증가**합니다.
   * 오판할 경우 진행 상황이 **1 단계 후퇴**합니다. (0 이하로는 내려가지 않음)
   * 특정 치명적인 이상현상(예: 새파란 손자국 추격, 그림자, 천장 붕괴 등)에 닿을 경우 **게임오버**가 되며, 진행 상태가 완전히 초기화되어 **0번 복도로 리셋**됩니다.
4. **탈출**: 출구 번호 **67**에 도달한 후 복도 끝에 생성되는 지상 계단을 끝까지 오르면 최종 탈출에 성공합니다.

---

## ✨ 핵심 특징 (Key Features)

* **에셋 프리(Asset-Free) 설계**: 이미지, 오디오 등 무거운 외부 에셋 파일을 일절 서버로부터 다운로드하지 않습니다. 모든 3D 텍스처(나무 마루, 포스터 등)와 사운드 이펙트(발소리, 빗소리, 비명 등)를 코드로 실시간 생성(Procedural Synthesis)하여 초기 로딩 시간이 거의 없습니다.
* **67가지 이상현상**: 플레이어의 보행 속도를 저하시키는 이변부터 갑작스러운 점프스케어, 물리 법칙이 뒤집히는 비유클리드 공간 연출까지 총 67가지의 다양한 이상현상이 준비되어 있습니다.
* **이변 도감 (Anomaly Dictionary)**: 메인 화면의 '이변 도감 보기'를 통해 활성화할 수 있으며, 기획된 모든 이상현상을 선택하여 해당 복도로 즉시 순간 이동하고 자유롭게 체험해 볼 수 있습니다.

---

## 🕹️ 조작 방법 (Controls)

* **시점 회전**: 마우스 이동 (마우스 클릭으로 화면 고정)
* **앞으로 이동**: `W`
* **뒤로 이동**: `S`
* **달리기**: `W` 또는 `S`를 누른 채 `Space` 유지
* **손전등 켜기/끄기**: `X`
* **문 열기 / 상호작용**: 마우스 좌클릭
* **일시 정지 / 메뉴**: `ESC`

---

## 🛠️ 실행 방법 (Local Run)

본 프로젝트는 초경량 웹 빌드 도구인 **Vite**를 사용하여 개발 및 실행됩니다. 

1. **종속성 패키지 설치**
   ```bash
   npm install
   ```

2. **개발 로컬 서버 실행**
   ```bash
   npm run dev
   ```
   * 실행 후 터미널에 표시되는 로컬 URL(기본값 `http://localhost:5173`)로 브라우저에서 접속하여 플레이할 수 있습니다.

3. **배포용 빌드**
   ```bash
   npm run build
   ```
